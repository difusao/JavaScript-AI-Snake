## AI-Snake
Famous snake game using Neural Networks and Genetic Algorithms.

## Neural Network
> Layers
- Inputs *24* (8 food sensor distances, 8 wall sensor distances, 8 tail sensor distance).
- Hidden *10*.
- OutPuts *4* (left, up, right, down)

> Function Activation
- ReLU

```sh

```
